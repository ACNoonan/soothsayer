## Conditional coverage — Kupiec + Christoffersen (F1_emp_regime)

Unconditional coverage (Kupiec POF) tests whether realized violation rate matches expected. Independence (Christoffersen) tests whether violations cluster — an institutional-grade disclosure. Conditional coverage (CC) combines both; `p_cc >= 0.05` means the forecaster is not rejected. Tests run per symbol's time series to respect weekend ordering; pooled p-values apply χ²(1 + n_symbols) under H0.
